# TodoPlus Project

TodoPlus is a small task management web app for practicing junior frontend development.

## Tech Stack

- React
- JavaScript
- HTML/CSS
- REST API
- SQLite for local demo data
- GitHub

## Features

- Task list and status filter
- Task detail editor
- Login state demo
- API loading and error states

## Implementation Notes

The project uses React hooks to manage UI state, separates components by feature, and calls API endpoints through a small client helper. The login state is a learning demo and should not be described as production OAuth.
