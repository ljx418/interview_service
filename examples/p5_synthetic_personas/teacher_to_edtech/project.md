# 合成项目资料：StudyPath Coach

> 本文件是 JobPilot P5 自动化验收使用的合成项目资料，不属于真实项目背调材料。

## 项目目标

帮助教师根据学生的练习记录生成学习路径、错题标签和下一步复习建议。

## 技术栈

- React、TypeScript、ECharts
- FastAPI、SQLite
- Markdown content pipeline

## 个人贡献

- 设计学习路径卡片和错题分类页面。
- 使用 ECharts 展示知识点掌握度和近期练习趋势。
- 编写 FastAPI 接口保存合成练习记录。
- 将课程讲义整理成 Markdown 知识点结构。

## 可验证证据

- 合成学生数据、页面截图、接口说明、Markdown 讲义样例。
- 不使用真实学生隐私数据。

## 风险与缺口

- 需要明确真实学生数据未进入项目。
- 需要补充工程测试和部署说明。
