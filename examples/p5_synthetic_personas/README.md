# P5 Synthetic Personas

These files are fictional acceptance materials for JobPilot P5.

They are designed to strengthen automated acceptance realism without using a real person's resume, project history, private JD, API key, account, or provider output. They must not be described as real personal-data validation.

## Personas

| Persona | Background | Target |
| --- | --- | --- |
| `ops_to_frontend` | Operations analyst transitioning to frontend | Junior Frontend Developer |
| `qa_to_fullstack` | Manual QA transitioning to full-stack QA automation | QA Automation / Full-stack Assistant |
| `teacher_to_edtech` | Teacher transitioning to education technology frontend | EdTech Frontend Developer |

Each persona includes:

- `resume.md`
- `project.md`
- `jd.md`
- `interview_brief.md`
