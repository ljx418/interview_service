## 3. 不得通过条件

出现以下任一项，P5 人工体验不得通过：

- 报告或截图泄露未授权完整简历、完整 JD、联系方式、账号、API Key 或私密链接；
- 默认触发真实外部 provider；
- 用户无法判断资料、JD、确认项、申请包或导出状态；
- 普通追问误触发 artifact 写入；
- blocking confirmation 未处理仍能导出正式申请材料；
- 把 fixture/examples 路径写成真实个人资料验收；
- 把 SaaS、ASR、会议平台、自动投递、MCP/CLI 或最终产品化写成 P5 已完成。
