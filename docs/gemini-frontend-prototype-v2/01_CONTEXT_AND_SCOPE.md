# 项目背景与范围

## 产品定位

JobPilot AI 是面向转行程序员的本地优先、免费开源 AI 求职 Agent 服务。默认入口是 Chatbox，核心目标是把简历、项目 README、JD 和面试 transcript 组织成可信、可确认、可导出的求职材料。

## 当前阶段

项目处于 P4 UX 体验强化阶段。P0-P3 已经证明本地 examples 路径、后端工具链、Chatbox 响应、推进台和截图报告可以跑通，但当前前端仍像工程验收控制台：

- 首屏视觉重心不够清楚；
- 用户需要理解工程状态才知道下一步；
- Workbench 与对话区虽然分离，但产物价值表达仍弱；
- provider、示例模式、我的资料模式还需要更自然的人类语言；
- 移动端可用，但不够顺手；
- 显式生成职业事实后，Workbench 计数和主体展示存在不一致。

## 目标用户

主要用户是转行程序员。用户希望快速知道：

- 我现在该先做什么；
- 系统是否会把我的资料发到外部模型；
- 当前已经生成了哪些求职材料；
- 哪些事实还需要确认；
- 什么时候可以导出申请材料。

## 本阶段目标

让用户不读开发文档，也能在本地 Chatbox 中完成：

1. 导入简历和项目资料；
2. 粘贴或讨论目标 JD；
3. 生成职业事实、岗位解析、匹配报告和申请包草稿；
4. 确认关键事实；
5. 导出材料；
6. 在 1200/1440/1600/1920 桌面、720 窄屏、390 移动端下顺手使用。

## 非目标

以下能力不属于 P4 原型范围：

- MCP Server、CLI；
- ASR、Whisper、会议平台、系统音频、视频解析；
- 自动海投、岗位数据源聚合、Offer 分析；
- SaaS 登录、多租户、Billing；
- 默认启用真实外部 provider；
- 使用真实个人资料做自动化验收；
- 把本地/mock 连续对话写成完整 provider-backed 自由智能聊天。
