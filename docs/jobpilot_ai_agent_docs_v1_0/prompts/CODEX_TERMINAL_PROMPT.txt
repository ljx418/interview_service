你现在是我的资深开源项目技术合伙人、Agent 架构师、产品工程负责人和代码审查者。请基于当前仓库中的文档，帮助我一起设计和实现 JobPilot AI。

项目方向：
JobPilot AI 是一个面向转行程序员的免费开源 AI 求职 Agent 服务。默认入口是极简 Chatbox，但核心不是前端页面，而是可被 Chatbox、CLI、HTTP API、MCP Client 或外部 Agent 调用的求职 Agent Service。

当前明确约束：
1. 不做注册登录系统。
2. 不做多租户 SaaS。
3. 不做付费订阅和 Billing。
4. 不做复杂 Dashboard、企业后台、求职漏斗大屏。
5. 不做自动海投机器人。
6. 不做隐蔽式面试作弊工具。
7. 不做面试官人脸识别、情绪识别或敏感属性分析。
8. 默认本地优先，数据保存到本地 workspace。
9. 前端只做极简 Chatbox + 文件上传 + 产物卡片 + 设置。
10. 核心能力必须工具化，可被 API/CLI/MCP 调用。
11. 所有对外材料必须基于用户事实库，不能编造经历、项目、数字、证书。
12. 不确定内容必须标注“需要用户确认”。
13. 面试实时助手默认只提供结构提示、项目推荐、时长提醒和复盘，不默认生成完整逐字答案。

目标用户：
转行程序员，包括自学编程者、Bootcamp 毕业生、非科班转码者，以及从运营、产品、设计、测试、技术支持、数据分析、教育、金融等方向转向开发岗位的人。

目标岗位：
Junior Frontend Developer、Junior Backend Developer、Full-stack Developer、Web Developer、AI Application Developer、QA Automation Engineer、Data Engineer 入门岗位、Developer Advocate 入门岗位。

MVP 核心闭环：
用户启动本地项目 → 创建 workspace → 打开 Chatbox → 上传简历和项目资料 → 生成职业事实库/技能证据库/技术项目卡 → 粘贴 JD → 生成岗位适合度分析 → 生成申请包 → 导出 Markdown/PDF → 生成面试准备包和故事卡 → 模拟面试 → 实时提示当前问题的回答结构 → 面试后复盘 → 生成下一步训练计划。

- 面试后复盘
- 轻量申请记录
- Agent Tool API

推荐技术栈：
- 前端：Vite + React + TypeScript + Tailwind CSS
- 后端：Python FastAPI
- 数据库：SQLite
- 文件：本地文件系统
- 向量索引：P0 可先简单实现，P1 接 Chroma/LanceDB/sqlite-vec
- LLM：OpenAI-compatible adapter + local model adapter
- ASR：P0 支持手动输入/转录文本 + ASR adapter 接口，P1 接本地 Whisper
- 导出：Markdown-first，P0 简单 PDF，P1 DOCX
