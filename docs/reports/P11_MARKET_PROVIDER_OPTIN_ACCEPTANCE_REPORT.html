P11 Market Provider Opt-in 自动化验收报告

验收结论：Level 1 通过

本报告验证 P11 本地/记录数据市场 provider opt-in 边界。它不代表真实市场 provider、全网 JD 搜索、招聘平台接入、ASR、MCP、自动投递或 SaaS 已通过。

目标架构与当前实现

目标架构：UI/CLI → FastAPI Market Provider Boundary → JobSearchRunService / Normalizer / SourceRefBinder → SQLite Workspace → Evidence Report。

当前实现：services/market/provider.py 提供 provider registry、policy gate、fixture/manual/public Level1 search run、snapshot 和 source refs；services/api/main.py 暴露 P11 API；apps/chatbox/src/main.tsx 与 services/cli/main.py 已读取 market provider 状态。
