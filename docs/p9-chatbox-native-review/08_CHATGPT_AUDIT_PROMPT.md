# ChatGPT 审计提示词

请审计 JobPilot AI P9 文档审查包。审计目标是判断当前 Agent 是否完整理解用户提出的前端优化诉求，并判断新版 PRD、前端体验规格和 HTML 审查页是否足以指导后续自动化开发与端到端验收。

请重点检查：

1. 是否明确当前仍是文档开发阶段，而非代码实现阶段。
2. 是否正确区分 P8.1 当前真实实现基线与 P9 目标设计。
3. 是否满足“不要很多向导卡片，中央 Chatbox 是主控入口”的要求。
4. 是否覆盖顶部服务中心、左侧求职态势图、中央 Chatbox、右侧产物台。
5. 是否覆盖通过 Chatbox 发起 JD 搜索、薪资城市汇总、资料采集、申请包生成、产物调整和招聘流程更新。
6. 是否正确处理全网 JD 搜索、ASR、真实 provider、MCP、Skill、招聘平台接入、自动投递等高风险边界。
7. HTML 审查页是否能让人类快速理解当前基线、目标设计、用户路线和验收门槛。
8. 是否存在事实性错误、过度承诺或 false-green 风险。

请输出：

- 结论：通过 / 不通过。
- 重大问题。
- 次要问题。
- 建议修订。
- 是否可以进入正式 P9 文档集开发。
- 是否可以进入代码开发。
